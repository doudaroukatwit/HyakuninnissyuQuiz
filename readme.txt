このアプリの説明書みたいなものです。
ランダムと順番(25首づつ)を選択して、答えの絵を選択するだけです。

アップデートは
メニューボタン→設定→アップデートを確認
または
右上のメニュー→設定→アップデートを確認
でお願いします。。

モードについて
・反復モード
　その問題が正解するまで、何度も繰り返します。
・スパルタモード
　問題数にかかわらず、間違えたら1問目に戻ります。ランダムの場合は同じ問題を繰り返すのではなく、また新しい問題になります。
